Crowdbox

Currently in private beta. Please don't tell anyone else about it.

Crowdbox is a Heroku-inspired hosting platform. The difference being that each app is funded by donations in the style of Wikipedia. Because of its donation-based funding model only public Github repositories can be hosted on it.

Crowdbox is built on Deis.

Getting Started

  • Install the Crowdbox client using pip: 
    $ pip install crowdbox
  • Login using your Github credentials: 
    $ crowdbox login
  • Go to the local version of your Github repository: 
    $ cd 
$ crowdbox create
Creating application... done, created opdemand-example--ruby--sinatra
  • Use `crowdbox deploy` to deploy your application: 
    $ crowdbox deploy
